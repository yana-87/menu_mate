# Menu Mate (献立提案アプリ)

## 1. アプリ概要と目的

料理のレパートリーがあるにも関わらず、「毎日の献立を考える時間」にストレスを感じているユーザーをターゲットとした、決断疲れを解消するための献立提案アプリ。

ボタン一つで「自分の作れる料理」だけからランダムに提案し、「迷う時間」をゼロにすることを目的とします。

## 2. 実装機能（Ver 1.0 - MVP）

- [ ] 料理の**登録**機能（名前と**ジャンル**を保存）
- [ ] 登録した料理の**一覧表示**機能
- [ ] 料理の**削除**機能
- [ ] **ガチャ機能**：登録メニューから**ランダム**に一つ提案

## 3. 技術スタック

- Python
- Flask
- Flask-SQLAlchemy (SQLite)
- Git / GitHub
- HTML / CSS

## 4. 将来的な拡張構想 (Ver 2.0)

- 提案アルゴリズムの改善（前日のジャンルを考慮した選出）
- 「和風、中華風」などの味付けジャンルの追加
